## 2024-05-14 - [Enhanced Debug Visibility]
**Learning:** For command-line tools without a GUI, the debug log *is* the interface. Translating raw hex codes into human-readable strings significantly reduces cognitive load and makes the tool more approachable for non-expert users.
**Action:** Always map common constants or codes to human-friendly labels in verbose/debug outputs.

## 2024-05-14 - [Documentation Hygiene]
**Learning:** Broken or outdated documentation links erode user trust and increase frustration during setup.
**Action:** Proactively verify external links when making changes to documentation or help messages.
